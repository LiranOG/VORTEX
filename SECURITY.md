# Security Policy

## Project Scope

VORTEX ETERNITY is a static, single-file HTML/JavaScript application.
It runs entirely client-side with no backend, no user data collection,
no authentication, and no network requests.

The attack surface is minimal by design.

## Supported Versions

| Version | Supported |
|---------|-----------|
| Latest (main) | ✅ |

## Reporting a Vulnerability

If you discover a security issue (e.g. a malicious dependency introduced
via CDN, or an XSS vector in the UI), please open a GitHub Issue marked
**[SECURITY]** or contact via LinkedIn:
[linkedin.com/in/liran-schwartz](https://linkedin.com/in/liran-schwartz)

Response expected within 7 days.
